5  MANOVA

MANOVA (Análisis Multivariado de Varianza) y MANCOVA (Análisis Multivariado de Covarianza)

Author

Samuel Solís Barrón

5.1 Objetivo

Comparar los vectores de medias de medias de tres o más grupos de manera simultánea, considerando múltiples variables de respuesta al mismo tiempo.

5.2 Datos utilizados

5.3.1 Preparación de los datos

El primer paso consiste en cargar el dataset y preparar la matriz de respuesta multivariada que utilizará el modelo MANOVA.

5.3.4 Verificación de supuestos del MANOVA

Los supuestos de MANOVA son similares en muchos aspectos a los de ANOVA, pero difieren en algunos puntos importantes. Antes de interpretar los resultados con plena validez estadística, es necesario verificar tres supuestos fundamentales.

5.3.4.1 Supuesto 1: Multinormalidad

Mientras que en ANOVA, el supuesto de normalidad se refería a una sola variable dependiente a lo largo de los niveles de la variable independiente, en MANOVA, además de considerar cada variable de manera univariada, el supuesto de normalidad también se aplica a una combinación lineal de variables dependientes. Esto es lo que se llama convenientemente normalidad multivariante. Con respecto al supuesto de normalidad de las variables dependientes consideradas por separado, como se hacía en ANOVA, se pueden obtener gráficos Q−Q o histogramas o, si se desea, una prueba formal de normalidad como la prueba de Shapiro–Wilk.

5.3.4.2 Supuesto 2: Homogeneidad de las matrices de varianzas-covarianzas

En ANOVA, se verifica el supuesto de homogeneidad de varianzas evaluando la hipótesis nula:

\[ H_0 : \sigma_1^2 = \sigma_2^2 = \sigma_3^2 = \sigma_4^2 \]

frente a la hipótesis alternativa de que existe alguna diferencia poblacional en las varianzas.

En MANOVA, se evalúa una hipótesis similar, pero la situación es más compleja porque ahora, además de las varianzas de cada variable dependiente, se tienen también las covarianzas. Dentro de cada nivel de la variable independiente existe una matriz de covarianza de las variables dependientes, cuya dimensión depende del número de variables dependientes en el análisis. Por ello, no basta con verificar igualdad de varianzas; también se debe verificar el supuesto de igualdad de covarianzas. Las varianzas y covarianzas se organizan en una matriz de covarianza, por lo que la hipótesis nula para las covarianzas es:

\[ H_0:\ \mathbf{\Sigma}_1 = \mathbf{\Sigma}_2 = \mathbf{\Sigma}_3 \]

Lo anterior indica que las matrices de covarianza poblacionales entre los niveles de la variable independiente son iguales, donde \(\mathbf{\Sigma}_1\) hasta \(\mathbf{\Sigma}_3\) representan las matrices de covarianza de cada grupo.

No es posible aplicar simplemente la prueba de Levene, ya que esta solo evalúa igualdad de varianzas y no de covarianzas. Se requiere, por tanto, un tipo de prueba diferente. La opción más utilizada es la prueba Box-M, que evalúa el supuesto de igualdad de matrices de covarianza. En esencia, la prueba busca detectar un posible desequilibrio entre las matrices de covarianza mediante el uso de determinantes.

# box_m() aplica la prueba de Box's M para evaluar si las matrices de
# varianza-covarianza son homogéneas entre los grupos (Species).
# H₀: las matrices de covarianza son iguales entre todos los grupos.
# Si p < 0.05 → rechazamos H₀ → las matrices NO son homogéneas.
# Este supuesto es más crítico que el de multinormalidad para la
# validez del MANOVA. Su violación puede considerarse usando
# estadísticos más robustos, como la Traza de Pillai.

box_m(iris[, 1:4], iris$Species)

5.3.4.3 Supuesto 3: Ausencia de valores atípicos extremos

Para verificar la ausencia de valores atípicos univariados o bivariados, se pueden producir histogramas y diagramas de dispersión tanto entre grupos como dentro de cada nivel de la variable independiente. En el caso multivariado, los valores atípicos pueden existir en ese espacio aunque sean extremadamente difíciles de detectar gráficamente. Por ello, se recomienda calcular las distancias de Mahalanobis, abreviadas frecuentemente como \(D^2\), para cada caso. Las distancias de Mahalanobis miden la distancia multivariada entre cada observación y la media multivariada del grupo (denominada centroide).

# Los valores atípicos multivariados se detectan mediante la
# distancia de Mahalanobis, que a diferencia de la distancia euclidiana,
# considera la correlación entre variables y las escalas distintas de cada una.
# Un punto es outlier multivariado si su distancia de Mahalanobis supera
# el percentil 97.5 de la distribución χ² con p grados de libertad
# (donde p = número de variables).
# Para este ejemplo, se asume que no existen valores atípicos extremos
# en el dataset iris, dado el origen controlado de los datos.

# mahalanobis() calcula la distancia D² de cada observación respecto al
# centroide (media multivariada) del grupo, tomando en cuenta la covarianza
# entre variables.

# Paso 1: seleccionar solo las variables numéricas
iris1 <- iris[, 1:4]

# Paso 2: calcular la media multivariada (centroide)
centro <- colMeans(iris1)       # Vector de medias de cada variable.

# Paso 3: calcular la matriz de covarianza
S <- cov(iris1)                 # Matriz de covarianza 4×4.

# Paso 4: calcular la distancia de Mahalanobis para cada observación
iris$D2 <- mahalanobis(iris1,   # Matriz de datos.
                       centro,  # Centroide del grupo.
                       S)       # Matriz de covarianza.

# Paso 5: identificar valores atípicos multivariados con el criterio chi-cuadrado
# Se usa qchisq con p = 0.001 y gl = número de variables (4).
# Observaciones con D² mayor al umbral se consideran atípicos multivariados.
umbral <- qchisq(0.999, df = ncol(iris1))

iris$outlier <- iris$D2 > umbral   # TRUE si es valor atípico multivariado.

# Paso 6: mostrar los casos atípicos detectados
iris[iris$outlier == TRUE, ]
[1] Sepal.Length Sepal.Width  Petal.Length Petal.Width  Species     
[6] D2           outlier

5.3.6 MANCOVA: incluir una covariable continua

# El MANCOVA extiende el MANOVA al incorporar una o más variables continuas
# como covariables, cuyo efecto se controla estadísticamente antes de
# evaluar el efecto del factor de interés.
#
# En este ejemplo, Sepal.Length actúa como covariable: queremos evaluar
# el efecto de Species sobre las demás variables morfológicas,
# eliminando la varianza explicada por la longitud del sépalo.
#
# IMPORTANTE: la covariable NO puede incluirse también en la matriz de respuesta.
# Si Sepal.Length aparece en ambos lados de la fórmula, la matriz de residuos
# pierde rango (rank 3 < 4) y summary.manova() lanza un error de singularidad.
#
# La solución es construir Y_mancova únicamente con las tres variables
# que NO se usan como covariable.

Y_mancova <- cbind(iris$Sepal.Width, iris$Petal.Length, iris$Petal.Width)

# manova() con covariable: "+ Sepal.Length" en el lado derecho de la fórmula
# indica que esta variable continua se incluye como covariable (no como factor).
# Es equivalente conceptualmente al ANCOVA, pero en el contexto multivariado.

fit_mancova <- manova(Y_mancova ~ Species + Sepal.Length, data = iris)

# summary.manova() muestra los resultados del MANCOVA.
# La tabla reporta el efecto de Species ajustado por Sepal.Length
# y el efecto independiente de Sepal.Length sobre el vector de respuesta.

summary.manova(fit_mancova)

5.3.7 Estadísticos multivariados disponibles en MANOVA

1) Lambda de Wilks, \(\Lambda = \dfrac{|\mathbf{E}|}{|\mathbf{H} + \mathbf{E}|}\). Wilks es lo que se conoce como un criterio inverso, lo que significa que valores pequeños, en lugar de grandes, constituyen evidencia en contra de la hipótesis nula multivariada. Recuérdese que en la razón \(F\) tradicional de ANOVA, son los valores grandes los que se cuentan como evidencia en contra de la nula. En \(\Lambda\), en cambio, la evidencia en contra de la nula se produce cuando \(\mathbf{H}\) es mucho mayor que \(\mathbf{E}\), donde \(\mathbf{H}\) es la matriz de hipótesis y \(\mathbf{E}\) la matriz de error. Considerando los casos extremos: si \(\mathbf{H}\) es mayor que cero y \(\mathbf{E}\) es igual a cero, entonces:

\[ \Lambda = \frac{|\mathbf{E}|}{|\mathbf{H} + \mathbf{E}|} = \frac{0}{|\mathbf{H} + 0|} = \frac{0}{|\mathbf{H}|} = 0 \]

Es decir, si \(\mathbf{H}\) explicara toda la variabilidad, \(\Lambda\) sería igual a 0. Por el contrario, si \(\mathbf{E}\) explicara toda la variabilidad y \(\mathbf{H}\) fuera nula:

\[ \Lambda = \frac{|\mathbf{E}|}{|\mathbf{H} + \mathbf{E}|} = \frac{|\mathbf{E}|}{|0 + \mathbf{E}|} = \frac{|\mathbf{E}|}{|\mathbf{E}|} = 1 \]

Cuando \(\mathbf{E}\) domina sobre \(\mathbf{H}\), los valores de \(\Lambda\) se acercan a 1, que es el máximo que puede alcanzar. El investigador, en cambio, espera valores pequeños, pues indican un posible efecto multivariado. Lambda de Wilks es una de las pruebas más utilizadas para evaluar efectos multivariados, cuenta con una larga tradición y es reportada por defecto en la mayoría del software estadístico.

2) Traza de Pillai, \(V^{(s)} = \text{tr}[(\mathbf{E} + \mathbf{H})^{-1}\mathbf{H}]\). La notación “tr” indica que se toma la traza de \((\mathbf{E} + \mathbf{H})^{-1}\mathbf{H}\). Dado que \(\mathbf{E} + \mathbf{H} = \mathbf{T}\), se puede reescribir \((\mathbf{E} + \mathbf{H})^{-1}\mathbf{H}\) como \(\mathbf{T}^{-1}(\mathbf{H})\). Así, el criterio de Pillai toma la traza de la matriz \(\mathbf{H}\) relativa a \(\mathbf{T}\). A diferencia de Wilks, donde se esperaban valores pequeños, en Pillai los valores grandes son indicativos de evidencia en contra de la hipótesis nula multivariada.

El criterio de Pillai también puede expresarse en función de los valores propios (eigenvalues):

\[ V^{(s)} = \sum_{i=1}^{s} \frac{\lambda_i}{1 + \lambda_i} \]

Esta interpretación cobra mayor relevancia en el contexto del análisis discriminante.

3) Raíz mayor de Roy, \(\theta = \dfrac{\lambda_1}{1 + \lambda_1}\). Aquí \(\lambda_1\) es el mayor de los valores propios extraídos. A diferencia de la traza de Pillai, que suma todos los valores propios como en \(\sum_{i=1}^{s} \dfrac{\lambda_i}{1 + \lambda_i}\), el criterio de Roy utiliza únicamente el mayor de los valores propios extraídos.

4) Traza de Lawley-Hotelling, \(U^{(s)} = \text{tr}(\mathbf{E}^{-1}\mathbf{H}) = \sum_{i=1}{s} \lambda_i\). Este criterio toma la traza de \(\mathbf{H}\) relativa a \(\mathbf{E}\).

# El argumento test = en summary.manova() permite seleccionar el estadístico
# con el que se evalúa la prueba global. Las cuatro opciones son:
#
# "Wilks"           → Lambda de Wilks (Λ): razón de determinantes
#                     |W| / |W + B|, donde W = matriz de dispersión intra-grupos
#                     y B = matriz de dispersión entre grupos. Es el estadístico
#                     por defecto; valores de Λ cercanos a 0 indican efecto fuerte.
#
# "Hotelling-Lawley" → Traza de Hotelling-Lawley: suma de los eigenvalores de
#                      W⁻¹B. Útil cuando el número de grupos es pequeño.
#
# "Pillai"          → Traza de Pillai: suma de los eigenvalores de
#                     B(W + B)⁻¹. Es el estadístico más robusto ante
#                     violaciones de los supuestos (normalidad u homogeneidad);
#                     se recomienda cuando estos supuestos no se cumplen.
#
# "Roy"             → Mayor eigenvalor de W⁻¹B (criterio de la raíz máxima).
#                     Es el más potente cuando solo existe una combinación lineal
#                     relevante, pero el menos robusto en general.
#
# Todos se convierten aproximadamente a un estadístico F de Fisher
# para obtener el valor p. En la mayoría de los casos llevan a la misma conclusión.
# Ante violaciones de supuestos, se prefiere la Traza de Pillai.

summary.manova(fit, test = "Pillai")

  • La mayor dificultad fue entender la diferencia entre usar cbind() con attach() y usar as.matrix() directamente en la fórmula. Ambos producen el mismo modelo, pero la segunda forma es más limpia y evita posibles conflictos de nombres en el entorno.

5.6 Aprendizaje

  • Aprendí que el MANOVA es una extensión natural del ANOVA para el caso donde tenemos múltiples variables de respuesta correlacionadas, y que usarlos de forma separada (ANOVA variable por variable) inflaría la tasa de error tipo I.

  • Comprendí que cuando el MANOVA resulta significativo, las pruebas post-hoc son ANOVAs individuales por variable, lo que permite identificar cuáles variables contribuyen a la diferencia detectada.

  • Aprendí la lógica detrás del MANCOVA: al agregar una covariable continua, se controla su efecto y se obtiene una evaluación más precisa del factor de interés (falta comprender más el tema).

  • Aunque ninguno de los supuestos se cumplió estrictamente en el dataset iris, se aprendió que la multinormalidad no es tan crítica con muestras grandes, pero que la homogeneidad de matrices de covarianza sí es importante para la validez del análisis, y que la Traza de Pillai es la respuesta estadística adecuada a esta violación.

  • Se logró solucionar la confusión con attach()/detach() y se adoptó la práctica de no usarlo para mantener el código más limpio y reproducible.

### Verificación de supuestos del MANOVA

Los supuestos de MANOVA son similares en muchos aspectos a los de ANOVA, pero difieren en algunos puntos importantes. Antes de interpretar los resultados con plena validez estadística, es necesario verificar tres supuestos fundamentales.

#### Supuesto 1: Multinormalidad

Mientras que en ANOVA, el supuesto de normalidad se refería a una sola variable dependiente a lo largo de los niveles de la variable independiente, en MANOVA, además de considerar cada variable de manera univariada, el supuesto de normalidad también se aplica a una combinación lineal de variables dependientes. Esto es lo que se llama convenientemente normalidad multivariante. Con respecto al supuesto de normalidad de las variables dependientes consideradas por separado, como se hacía en ANOVA, se pueden obtener gráficos Q−Q o histogramas o, si se desea, una prueba formal de normalidad como la prueba de Shapiro–Wilk.

```{r}
# head() muestra las primeras 6 filas del dataset para confirmar
# su estructura antes de aplicar cualquier prueba estadística.
# Permite verificar que las columnas numéricas están bien definidas
# y que no hay valores inesperados en las primeras observaciones.

head(iris)

```

```{r}
# mshapiro_test() aplica la prueba de Shapiro-Wilk multivariada
# (extensión de la prueba univariada de Shapiro-Wilk al caso multivariado).
# H₀: los datos siguen una distribución multinormal.
# Si p < 0.05 → rechazamos H₀ → los datos NO tienen distribución multinormal.
# Nota importante: la falta de multinormalidad no invalida el MANOVA
# cuando los tamaños de muestra son grandes, gracias al teorema del
# límite central multivariado (n ≥ 20 por grupo suele ser suficiente).

mshapiro_test(iris[, 1:4])

```

#### Supuesto 2: Homogeneidad de las matrices de varianzas-covarianzas

En ANOVA, se verifica el supuesto de homogeneidad de varianzas evaluando la 
hipótesis nula:

$$
H_0 : \sigma_1^2 = \sigma_2^2 = \sigma_3^2 = \sigma_4^2
$$

frente a la hipótesis alternativa de que existe alguna diferencia poblacional 
en las varianzas.

En MANOVA, se evalúa una hipótesis similar, pero la situación es más compleja 
porque ahora, además de las varianzas de cada variable dependiente, se tienen 
también las **covarianzas**. Dentro de cada nivel de la variable independiente 
existe una matriz de covarianza de las variables dependientes, cuya dimensión 
depende del número de variables dependientes en el análisis. Por ello, no basta 
con verificar igualdad de varianzas; también se debe verificar el supuesto de 
**igualdad de covarianzas**. Las varianzas y covarianzas se organizan en una 
**matriz de covarianza**, por lo que la hipótesis nula para las covarianzas es:

$$
H_0:\ \mathbf{\Sigma}_1 = \mathbf{\Sigma}_2 = \mathbf{\Sigma}_3
$$

Lo anterior indica que las matrices de covarianza poblacionales entre los niveles  de la variable independiente son iguales, donde $\mathbf{\Sigma}_1$ hasta $\mathbf{\Sigma}_3$ representan las matrices de covarianza de cada grupo.

No es posible aplicar simplemente la prueba de Levene, ya que esta solo evalúa igualdad de varianzas y no de covarianzas. Se requiere, por tanto, un tipo de prueba diferente. La opción más utilizada es la **prueba Box-M**, que evalúa  el supuesto de igualdad de matrices de covarianza. En  esencia, la prueba busca detectar un posible **desequilibrio** entre las matrices de covarianza mediante el uso de determinantes.

```{r}
# box_m() aplica la prueba de Box's M para evaluar si las matrices de
# varianza-covarianza son homogéneas entre los grupos (Species).
# H₀: las matrices de covarianza son iguales entre todos los grupos.
# Si p < 0.05 → rechazamos H₀ → las matrices NO son homogéneas.
# Este supuesto es más crítico que el de multinormalidad para la
# validez del MANOVA. Su violación puede considerarse usando
# estadísticos más robustos, como la Traza de Pillai.

box_m(iris[, 1:4], iris$Species)

```

#### Supuesto 3: Ausencia de valores atípicos extremos

Para verificar la ausencia de valores atípicos univariados o bivariados, se pueden  producir histogramas y diagramas de dispersión tanto entre grupos como dentro de  cada nivel de la variable independiente. En el caso multivariado, los valores  atípicos pueden existir en ese espacio aunque sean extremadamente difíciles de  detectar gráficamente. Por ello, se recomienda calcular las **distancias de Mahalanobis**, abreviadas frecuentemente como $D^2$, para cada caso. Las  distancias de Mahalanobis miden la distancia multivariada entre cada observación y la media multivariada del grupo (denominada centroide).

```{r}
# Los valores atípicos multivariados se detectan mediante la
# distancia de Mahalanobis, que a diferencia de la distancia euclidiana,
# considera la correlación entre variables y las escalas distintas de cada una.
# Un punto es outlier multivariado si su distancia de Mahalanobis supera
# el percentil 97.5 de la distribución χ² con p grados de libertad
# (donde p = número de variables).
# Para este ejemplo, se asume que no existen valores atípicos extremos
# en el dataset iris, dado el origen controlado de los datos.

# mahalanobis() calcula la distancia D² de cada observación respecto al
# centroide (media multivariada) del grupo, tomando en cuenta la covarianza
# entre variables.

# Paso 1: seleccionar solo las variables numéricas
iris1 <- iris[, 1:4]

# Paso 2: calcular la media multivariada (centroide)
centro <- colMeans(iris1)       # Vector de medias de cada variable.

# Paso 3: calcular la matriz de covarianza
S <- cov(iris1)                 # Matriz de covarianza 4×4.

# Paso 4: calcular la distancia de Mahalanobis para cada observación
iris$D2 <- mahalanobis(iris1,   # Matriz de datos.
                       centro,  # Centroide del grupo.
                       S)       # Matriz de covarianza.

# Paso 5: identificar valores atípicos multivariados con el criterio chi-cuadrado
# Se usa qchisq con p = 0.001 y gl = número de variables (4).
# Observaciones con D² mayor al umbral se consideran atípicos multivariados.
umbral <- qchisq(0.999, df = ncol(iris1))

iris$outlier <- iris$D2 > umbral   # TRUE si es valor atípico multivariado.

# Paso 6: mostrar los casos atípicos detectados
iris[iris$outlier == TRUE, ]

```

### MANCOVA: incluir una covariable continua

```{r}
# El MANCOVA extiende el MANOVA al incorporar una o más variables continuas
# como covariables, cuyo efecto se controla estadísticamente antes de
# evaluar el efecto del factor de interés.
#
# En este ejemplo, Sepal.Length actúa como covariable: queremos evaluar
# el efecto de Species sobre las demás variables morfológicas,

# Si Sepal.Length aparece en ambos lados de la fórmula, la matriz de residuos

#
# La solución es construir Y_mancova únicamente con las tres variables
# que NO se usan como covariable.

# manova() con covariable: "+ Sepal.Length" en el lado derecho de la fórmula
# indica que esta variable continua se incluye como covariable (no como factor).
# Es equivalente conceptualmente al ANCOVA, pero en el contexto multivariado.

fit_mancova <- manova(Y_mancova ~ Species + Sepal.Length, data = iris)

# summary.manova() muestra los resultados del MANCOVA.
# La tabla reporta el efecto de Species ajustado por Sepal.Length
# y el efecto independiente de Sepal.Length sobre el vector de respuesta.

summary.manova(fit_mancova)
```

### Estadísticos multivariados disponibles en MANOVA

**1) Lambda de Wilks**, $\Lambda = \dfrac{|\mathbf{E}|}{|\mathbf{H} + \mathbf{E}|}$.  Wilks es lo que se conoce como un **criterio inverso**, lo que significa que  valores pequeños, en lugar de grandes, constituyen evidencia en contra de la  hipótesis nula multivariada. Recuérdese que en la razón $F$ tradicional de ANOVA, son los valores grandes los que se cuentan como evidencia en contra de la nula. En $\Lambda$, en cambio, la evidencia en contra de la nula se produce cuando $\mathbf{H}$ es mucho mayor que $\mathbf{E}$, donde $\mathbf{H}$ es la matriz de **hipótesis** y $\mathbf{E}$ la matriz de **error**. Considerando los casos extremos: si $\mathbf{H}$ es mayor que cero y $\mathbf{E}$ es igual a cero, entonces:

$$
\Lambda = \frac{|\mathbf{E}|}{|\mathbf{H} + \mathbf{E}|} = \frac{0}{|\mathbf{H} + 0|} = \frac{0}{|\mathbf{H}|} = 0
$$

Es decir, si $\mathbf{H}$ explicara toda la variabilidad, $\Lambda$ sería igual  a 0. Por el contrario, si $\mathbf{E}$ explicara toda la variabilidad y $\mathbf{H}$ fuera nula:

$$
\Lambda = \frac{|\mathbf{E}|}{|\mathbf{H} + \mathbf{E}|} = \frac{|\mathbf{E}|}{|0 + \mathbf{E}|} = \frac{|\mathbf{E}|}{|\mathbf{E}|} = 1
$$

Cuando $\mathbf{E}$ domina sobre $\mathbf{H}$, los valores de $\Lambda$ se  acercan a 1, que es el máximo que puede alcanzar. El investigador, en cambio, espera valores pequeños, pues indican un posible efecto multivariado. Lambda de Wilks es una de las pruebas más utilizadas para evaluar efectos multivariados, cuenta con una larga tradición y es reportada por defecto en la mayoría del software estadístico.

**2) Traza de Pillai**, $V^{(s)} = \text{tr}[(\mathbf{E} + \mathbf{H})^{-1}\mathbf{H}]$. La notación "tr" indica que se toma la traza de $(\mathbf{E} + \mathbf{H})^{-1}\mathbf{H}$. Dado que $\mathbf{E} + \mathbf{H} = \mathbf{T}$, se puede reescribir $(\mathbf{E} + \mathbf{H})^{-1}\mathbf{H}$ como $\mathbf{T}^{-1}(\mathbf{H})$. Así, el criterio de Pillai toma la traza de la matriz $\mathbf{H}$ relativa a $\mathbf{T}$. A diferencia de Wilks, donde se esperaban valores pequeños, en Pillai los valores **grandes** son indicativos de evidencia en contra de la hipótesis nula multivariada.

El criterio de Pillai también puede expresarse en función de los **valores propios** (*eigenvalues*):

$$
V^{(s)} = \sum_{i=1}^{s} \frac{\lambda_i}{1 + \lambda_i}
$$

Esta interpretación cobra mayor relevancia en el contexto del análisis discriminante.

**3) Raíz mayor de Roy**, $\theta = \dfrac{\lambda_1}{1 + \lambda_1}$. Aquí $\lambda_1$ es el mayor de los valores propios extraídos. A diferencia de la traza de Pillai, que suma todos los valores propios como en $\sum_{i=1}^{s} \dfrac{\lambda_i}{1 + \lambda_i}$, el criterio de Roy utiliza únicamente el mayor de los valores propios extraídos.

**4) Traza de Lawley-Hotelling**, $U^{(s)} = \text{tr}(\mathbf{E}^{-1}\mathbf{H}) = \sum_{i=1}{s} \lambda_i$. Este criterio toma la traza de $\mathbf{H}$ relativa a $\mathbf{E}$.

```{r}
# El argumento test = en summary.manova() permite seleccionar el estadístico
# con el que se evalúa la prueba global. Las cuatro opciones son:
#
# "Wilks"           → Lambda de Wilks (Λ): razón de determinantes
#                     |W| / |W + B|, donde W = matriz de dispersión intra-grupos
#                     y B = matriz de dispersión entre grupos. Es el estadístico
#                     por defecto; valores de Λ cercanos a 0 indican efecto fuerte.
#
# "Hotelling-Lawley" → Traza de Hotelling-Lawley: suma de los eigenvalores de
#                      W⁻¹B. Útil cuando el número de grupos es pequeño.
#
# "Pillai"          → Traza de Pillai: suma de los eigenvalores de
#                     B(W + B)⁻¹. Es el estadístico más robusto ante
#                     violaciones de los supuestos (normalidad u homogeneidad);
#                     se recomienda cuando estos supuestos no se cumplen.
#
# "Roy"             → Mayor eigenvalor de W⁻¹B (criterio de la raíz máxima).
#                     Es el más potente cuando solo existe una combinación lineal
#                     relevante, pero el menos robusto en general.
#
# Todos se convierten aproximadamente a un estadístico F de Fisher
# para obtener el valor p. En la mayoría de los casos llevan a la misma conclusión.
# Ante violaciones de supuestos, se prefiere la Traza de Pillai.

summary.manova(fit, test = "Pillai")

```

#### Multinormalidad multivariada

No obstante, esto no invalida el MANOVA por dos razones: (1) el dataset cuenta con 50 observaciones por grupo (n = 150 total), un tamaño muestral que activa el teorema del límite central multivariado, haciendo que el MANOVA sea robusto ante desviaciones moderadas de la normalidad; y (2) el uso de la Traza de Pillai como estadístico alternativo ofrece mayor robustez ante esta violación.

#### Homogeneidad de matrices de varianza-covarianza

```{r}

# H₀: las matrices de covarianza son iguales entre las tres especies.

box_m(iris[, 1:4], iris$Species)

```

**Interpretación:** El resultado de la prueba de Box's M es significativo (p < 0.05), lo que indica que **las matrices de varianza-covarianza no son homogéneas** entre las tres especies. Este supuesto es el más importante para la validez del MANOVA, porque su violación puede inflar el error tipo I. En la práctica, la prueba de Box's M es muy sensible al tamaño de muestra y a la no-normalidad, por lo que una violación no necesariamente descalifica el análisis. La recomendación en este caso es reportar los resultados usando la **Traza de Pillai** en lugar de la Lambda de Wilks, ya que el estadístico de Pillai es el más robusto ante la heterogeneidad de matrices de covarianza.
